Design tip:

You can select two ways of viewing design iterations. The List view tab presents them in them in the order created. The Tree view shows how each iteration depends on previous ones. Pick the one that makes most sense to you!

In either view, you can use the up and down arrow keys to scroll through the design iterations while viewing each in the Preview window. Navigate up and down through the views with arrow keys.
